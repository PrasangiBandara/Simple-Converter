# Simple-Converter create using JavaFX

Conversions:
  1. Convert to meter (m)
  2. Convert to kilo meter (km)
  3. Convert to Volume(V)	
  4. Convert to Celsius(*C)
  5. Convert to Fahrenheit(*F)
  6. Convert to INR
